# Usage
<!-- usage -->
```sh-session
$ npm install -g timetracker
$ tt COMMAND
running command...
$ tt (-v|--version|version)
timetracker/0.0.0 linux-x64 node-v12.16.1
$ tt --help [COMMAND]
USAGE
  $ tt COMMAND
...
```
